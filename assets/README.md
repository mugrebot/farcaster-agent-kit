# Assets Directory

This directory contains images and other assets for the Farcaster Agent Kit.

## $CLANKIT Network Token Logo

Put your CLANKIT logo here as:
```
assets/clankit-logo.png
```

### Requirements:
- **Format**: PNG (recommended) or JPG
- **Size**: 400x400px minimum (square aspect ratio)
- **Quality**: High resolution for best results
- **Style**: Should represent the agent network/collective

### Usage:
1. Add your logo to this directory
2. Run `npm run pin-image` to pin it to IPFS
3. Run `npm run launch-network-token` to launch with your logo

### IPFS Integration:
The logo gets pinned to IPFS for decentralized hosting:
- Generates `ipfs://` URL automatically
- Stored permanently on IPFS network
- Accessible via any IPFS gateway
- Used in @clanker token launch

### Example Logo Concepts:
- Robotic/AI themed imagery
- Network/connection visuals
- Farcaster purple (#8465CB) color scheme
- Clean, modern design
- Recognizable as a collective/network token
